【MS SQL】日期函數（Date and Time Functions）
Created 2020-06-14| Updated 2020-06-14|MS SQLBasic
Word count:903|Reading time: 3 min
| Post View:

前言

SQL 內建函數有多種，而最常使用的四大函數分別為聚合函數（Aggregate Functions）& 字串函數（String Functions）& 數值函數（Mathematical Functions）& 日期函數（Date and Time Functions）。

正文

前面介紹關於字串及數值的函數，當然也不能少了日期函數，在 Data 的世界裡，個人認為最難處理的就是日期了，總是有讓人意想不到的情況…

本文會介紹 不會介紹
YEAR CURRENT_TIMESTAMP
MONTH TODATETIMEOFFSET
DAY TIMEFROMPARTS
GETDATE SWITCHOFFSET
GETUTCDATE SMALLDATETIMEFROMPARTS
SYSDATETIME DATETIMEOFFSETFROMPARTS
SYSUTCDATETIME EOMONTH
SYSDATETIMEOFFSET DATETIMEFROMPARTS
DATEPART DATETIME2FROMPARTS
DATENAME DATEFROMPARTS
DATEADD DATEDIFF

@@DATEFIRST

YEAR

傳回指定日期中年的部分，回傳 int

📕 語法：YEAR(date)

取得 2020/6/14 年的部分 SELECT YEAR('2020/6/14')

MONTH

傳回指定日期中月的部分，回傳 int

📕 語法：MONTH(date)

取得 2020/6/14 月的部分 SELECT MONTH('2020/6/14')

DAY

傳回指定日期日的部分，回傳 int

📕 語法：DAY(datetime)

取得 2020/6/14 日的部分 SELECT DAY('2020/6/14')

GETDATE

用來取得資料庫系統時間，通常使用於取得當下日期或時間，回傳 datetime Type

📕 語法：GETDATE()

取得當下資料庫系統時間 SELECT GETDATE()

GETUTCDATE

取得目前 UTC 國際標準時間，回傳 datetime Type

📕 語法：GETUTCDATE()

取得國際標準時間 SELECT GETUTCDATE()

SYSDATETIME

與 GETDATE 一樣是取得資料庫系統時間，回傳比 GETDATE 還多的秒數有效位數，回傳 datetime2(7) Type

📕 語法：SYSDATETIME

SYSDATETIME 比 GETDATE 多出 4 位秒數 SELECT SYSDATETIME() AS 'SYSDATETIME', GETDATE() AS 'GETDATE'

SYSUTCDATETIME

與 GETUTCDATE 相同，用來取得 UTC 國際標準時間，倆差別於 SYSUTCDATETIME 比 GETUTCDATE 多出較多的秒數有效位數，回傳 datetime2

📕 語法：SYSUTCDATETIME()

SYSUTCDATETIME 比 GETUTCDATE 多出 4 位秒數

SYSDATETIMEOFFSET

取得當下資料庫系統時間且包含時區，回傳 datetimeoffset(7)

📕 語法：SYSDATETIMEOFFSET()

取得當下時間包含時區 SELECT SYSDATETIMEOFFSET()

DATEPART

除了 YEAR、MONTH、DAY 還可使用 DATEPART 來取得指定日期指定的部分，DATEPART 能取得更多的部分如：時、分、秒等。回傳整數型態

📕 語法：DATEPART(datepart, date)

DATENAME

DATENAME 與 DATEPART 基本上是相同的，不同的是 DATENAME 會回傳字串型態。

📕 語法：DATENAME(datepart, date)

取得當下日期年的部分在加上字串 1 DATEPART 與 DATENAME 差別顯而易見

SELECT DATENAME(YEAR, GETDATE())+'1' AS 'DATENAME', DATEPART(YEAR, GETDATE())+'1' AS 'DATEPART'

DATEADD

對指定的日期累加值，如取得當下日期增加一年

📕 語法：DATEADD(datepart, num, date)

ISDATE

用來判斷是否為有效日期或時間，有效返回 1 ，無效則返回 0

📕 語法：ISDATE(expression)

分別判斷 2020-12-31 & 2020/12/31 & 123 是否為有效日期
SELECT ISDATE('2020-12-31'), ISDATE('2020/12/31'), ISDATE('123')

結語

使用日期函數取得 YEAR、MONTH、DAY 時要注意，若未提供年月日，僅提供時間的部分，SQL SERVER 會回傳 1900/1/1。
