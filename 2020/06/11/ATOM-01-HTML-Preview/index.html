前言

ATOM 自帶即時預覽，使用快捷鍵 Ctrl＋Shift＋M 即可看到切出的預覽視窗，缺點是未能帶上 CSS 樣式，若需要帶上 CSS 樣式得另外安裝外掛套件。

原生預覽

先來看看原生預覽效果

安裝 ATOM-HTML-PREVIEW 套件

📌 Step 1
指向 File 選擇 Settings

📌 Step 2
選擇 Install 並填入外掛名稱「ATOM-HTML-PREVIEW」，找到後點擊 Install

📌 分別建立 index.html & style.css

使用快捷鍵 Ctrl＋Shift＋H 開啟 Preview 預覽效果，或者在 index.html 視窗空白處執行右鍵，選擇 Preview HTML

可看到右邊多切出一個 Preview 視窗，且也有吃到 Include 的 style.css 樣式

結語

如此一來就可以邊寫邊預覽結果，不用再開啟瀏覽器，挺方便的！
